# 🎓 Student Management System

The Student Management System is a modern web-based application designed to manage student records efficiently through a professional and user-friendly interface.

This system allows users to add, update, manage and delete student information in a secure MySQL database using a Flask backend. The website provides a smooth cinematic interface with responsive design and modern animations for a premium user experience.

# How It Works

- Users enter student details through an interactive form.
- The Flask backend processes the submitted data.
- Student records are stored securely in the MySQL database.
- All data can be managed dynamically from the website interface.
- The system supports real-time form handling and database operations.

#  Features

- Add Student Records
- Update Student Information
- Delete Student Data
- MySQL Database Integration
- Secure Environment Variables (.env)
- Responsive Cinematic UI
- Smooth User Experience
- Professional Dashboard Interface
- Dynamic Form Management
